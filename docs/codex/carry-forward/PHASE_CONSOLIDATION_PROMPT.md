# Control Prompt: Phase Consolidation and Replay Prep

Use this prompt in control context at the end of a phase to prepare for future full replay from scratch.

Goal
- Consolidate lessons from all tracks into a clean, versioned prompt-system baseline for future reconstruction.

Inputs
- All track synthesis artifacts
- All control decisions and adversarial reviews
- Current codebase health notes and technical debt list

Instructions
1. Consolidate cross-track rules into a single control summary.
2. Separate durable rules from temporary workaround rules.
3. Produce a replay readiness checklist that can rebuild the codebase without historical cruft.
4. Define baseline prompt/template updates for next generation (for example, CODEX v2).
5. Define migration plan from current prompt library to next-generation prompt library.
6. Keep this step non-disruptive to active delivery work.

Output format
- Consolidated lessons register
- Durable versus temporary rules
- Replay readiness checklist
- Next-generation prompt-library upgrade plan
- Risk register and mitigations
